Physics Data Booklet Formulas (IB DP)

Mathematical equations

$$A = \pi r^2$$
$$A = \frac{1}{2}(bh)$$
$$C = 2\pi r$$
$$V = lwh$$
$$V = \pi r^2 h$$
$$V = Ah$$
$$V = \frac{4}{3}\pi r^3$$
$$A = 2\pi rh$$
$$A_H = A \cos\theta$$
$$A_v = A \sin\theta$$
$$\tan\theta = \frac{\sin\theta}{\cos\theta}$$
$$\sin^2\theta + \cos^2\theta = 1$$

Uncertainties

If $y = a \pm b$, then:

$$\Delta y = \Delta a + \Delta b$$

If $y = \frac{ab}{c}$, then:

$$\frac{\Delta y}{y} = \frac{\Delta a}{a} + \frac{\Delta b}{b} + \frac{\Delta c}{c}$$

If $y = a^n$, then:

$$\frac{\Delta y}{y} = |n \frac{\Delta a}{a}|$$

A. Space, time and motion

A.1 Kinematics

$$s = \frac{u+v}{2}t$$
$$v = u + at$$
$$s = ut + \frac{1}{2}at^2$$
$$v^2 = u^2 + 2as$$

A.2 Forces and momentum

$$F_f \le \mu_s F_N$$
$$F_f = \mu_d F_N$$
$$F_H = -kx$$
$$F_d = 6\pi\eta rv$$
$$F_b = \rho Vg$$
$$F_g = mg$$
$$p = mv$$
$$J = F\Delta t$$
$$F = ma = \frac{\Delta p}{\Delta t}$$
$$a = \frac{v^2}{r} = \omega^2 r = \frac{4\pi^2 r}{T^2}$$
$$v = \frac{2\pi r}{T} = \omega r$$

A.3 Work, energy and power

$$W = Fs \cos\theta$$
$$E_k = \frac{1}{2}mv^2 = \frac{p^2}{2m}$$
$$\Delta E_p = mg\Delta h$$
$$E_H = \frac{1}{2}k\Delta x^2$$
$$P = \frac{\Delta W}{\Delta t} = Fv$$
$$\eta = \frac{\text{useful work out}}{\text{total work in}} = \frac{\text{useful power out}}{\text{total power in}}$$

A.4 Rigid body mechanics (AHL)

$$\tau = Fr \sin\theta$$
$$\Delta\theta = \frac{\omega_f + \omega_i}{2}t$$
$$\omega_f = \omega_i + \alpha t$$
$$\Delta\theta = \omega_i t + \frac{1}{2}\alpha t^2$$
$$\omega_f^2 = \omega_i^2 + 2\alpha\Delta\theta$$
$$I = \Sigma mr^2$$
$$\tau = I\alpha$$
$$L = I\omega$$
$$\Delta L = \tau\Delta t$$
$$\Delta L = \Delta(I\omega)$$
$$E_k = \frac{1}{2}I\omega^2 = \frac{L^2}{2I}$$

A.5 Galilean and special relativity (AHL)

$$x' = x - vt$$
$$t' = t$$
$$u' = u - v$$
$$\gamma = \frac{1}{\sqrt{1 - \frac{v^2}{c^2}}}$$
$$x' = \gamma(x - vt)$$
$$t' = \gamma(t - \frac{vx}{c^2})$$
$$u' = \frac{u - v}{1 - \frac{uv}{c^2}}$$
$$(\Delta s)^2 = (c\Delta t)^2 - \Delta x^2$$
$$\Delta t = \gamma\Delta t_0$$
$$L = \frac{L_0}{\gamma}$$
$$\tan\theta = \frac{v}{c}$$

B. The particulate nature of matter

B.1 Thermal energy transfers

$$\rho = \frac{m}{V}$$
$$\overline{E_k} = \frac{3}{2}k_B T$$
$$Q = mc\Delta T$$
$$Q = mL$$
$$\frac{\Delta Q}{\Delta t} = kA\frac{\Delta T}{\Delta x}$$
$$L = \sigma A T^4$$
$$b = \frac{L}{4\pi d^2}$$
$$\lambda_{max}T = 2.9 \times 10^{-3} \text{mK}$$

B.2 Greenhouse effect

$$\text{albedo} = \frac{\text{total scattered power}}{\text{total incident power}}$$
$$\text{emissivity} = \frac{\text{power radiated per unit area}}{\sigma T^4}$$

B.3 Gas laws

$$P = \frac{F}{A}$$
$$n = \frac{N}{N_A}$$
$$\frac{PV}{T} = \text{constant}$$
$$PV = nRT = Nk_B T$$
$$P = \frac{1}{3}\rho \overline{v^2}$$
$$U = \frac{3}{2}nRT = \frac{3}{2}Nk_B T$$

B.4 Thermodynamics (AHL)

$$Q = \Delta U + W$$
$$W = P\Delta V$$
$$\Delta U = \frac{3}{2}nR\Delta T = \frac{3}{2}Nk_B\Delta T$$
$$\Delta S = \frac{\Delta Q}{T}$$
$$S = k_B \ln\Omega$$
$$PV^{\frac{5}{3}} = \text{constant}$$
$$\eta = \frac{\text{useful work}}{\text{input energy}}$$
$$\eta_{Carnot} = 1 - \frac{T_c}{T_h}$$

B.5 Current and circuits

$$I = \frac{\Delta q}{\Delta t}$$
$$V = \frac{W}{q}$$
$$R = \frac{V}{I}$$
$$\rho = \frac{RA}{L}$$
$$P = IV = I^2 R = \frac{V^2}{R}$$

Series circuits

$$I = I_1 = I_2 = \dots$$
$$V = V_1 + V_2 + \dots$$
$$R_s = R_1 + R_2 + \dots$$

Parallel circuits

$$I = I_1 + I_2 + \dots$$
$$V = V_1 = V_2 = \dots$$
$$\frac{1}{R_p} = \frac{1}{R_1} + \frac{1}{R_2} + \dots$$

EMF

$$\epsilon = I(R+r)$$

C. Wave behaviour

C.1 Simple harmonic motion

$$a = -\omega^2 x$$
$$T = \frac{1}{f} = \frac{2\pi}{\omega}$$
$$T = 2\pi\sqrt{\frac{m}{k}}$$
$$T = 2\pi\sqrt{\frac{l}{g}}$$

AHL

$$x = x_0 \sin(\omega t + \phi)$$
$$v = \omega x_0 \cos(\omega t + \phi)$$
$$v = \pm\omega\sqrt{x_0^2 - x^2}$$
$$E_T = \frac{1}{2}m\omega^2 x_0^2$$
$$E_p = \frac{1}{2}m\omega^2 x^2$$

C.2 Wave model

$$v = f\lambda = \frac{\lambda}{T}$$

C.3 Wave phenomena

$$\frac{n_1}{n_2} = \frac{\sin\theta_2}{\sin\theta_1} = \frac{v_2}{v_1}$$

Constructive interference:

$$\text{path difference} = n\lambda$$

Destructive interference:

$$\text{path difference} = (n + \frac{1}{2})\lambda$$

$$s = \frac{\lambda D}{d}$$

AHL

$$\theta = \frac{\lambda}{b}$$
$$n\lambda = d\sin\theta$$

C.5 Doppler effect

$$\frac{\Delta f}{f} = \frac{\Delta\lambda}{\lambda} \approx \frac{v}{c}$$

AHL

Moving source:

$$f' = f(\frac{v}{v \pm u_s})$$

Moving observer:

$$f' = f(\frac{v \pm u_o}{v})$$

D. Fields

D.1 Gravitational fields

$$F = G\frac{m_1 m_2}{r^2}$$
$$g = \frac{F}{m} = G\frac{M}{r^2}$$

AHL

$$E_p = -G\frac{m_1 m_2}{r}$$
$$V_g = -G\frac{M}{r}$$
$$g = -\frac{\Delta V_g}{\Delta r}$$
$$W = m\Delta V_g$$
$$v_{esc} = \sqrt{\frac{2GM}{r}}$$
$$v_{orbital} = \sqrt{\frac{GM}{r}}$$

D.2 Electric and magnetic fields

$$F = k\frac{q_1 q_2}{r^2} \text{ where } k = \frac{1}{4\pi\epsilon_0}$$
$$E = \frac{F}{q}$$
$$E = \frac{V}{d}$$

AHL

$$E_p = k\frac{q_1 q_2}{r}$$
$$V_e = \frac{kQ}{r}$$
$$E = -\frac{\Delta V_e}{\Delta r}$$
$$W = q\Delta V_e$$

D.3 Motion in electromagnetic fields

$$F = qvB\sin\theta$$
$$F = BIL\sin\theta$$
$$\frac{F}{L} = \mu_0 \frac{I_1 I_2}{2\pi r}$$

D.4 Induction

$$\Phi = BA\cos\theta$$
$$\epsilon = -N\frac{\Delta\Phi}{\Delta t}$$
$$\epsilon = BvL$$

E. Nuclear and quantum physics

E.1 Structure of the atom

$$E = hf$$

AHL

$$R = R_0 A^{\frac{1}{3}}$$
$$E = -\frac{13.6}{n^2}\text{eV}$$
$$mvr = \frac{nh}{2\pi}$$

E.2 Quantum physics (AHL)

$$E_{max} = hf - \Phi$$
$$\lambda = \frac{h}{p}$$
$$\lambda_f - \lambda_i = \Delta\lambda = \frac{h}{m_e c}(1 - \cos\theta)$$

E.3 Radioactive decay

$$E = mc^2$$

AHL

$$N = N_0 e^{-\lambda t}$$
$$A = \lambda N = \lambda N_0 e^{-\lambda t}$$
$$T_{\frac{1}{2}} = \frac{\ln 2}{\lambda}$$

E.5 Fusion and stars

$$d(\text{parsec}) = \frac{1}{p(\text{arc-second})}$$
